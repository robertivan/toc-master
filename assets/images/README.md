## Image Guidelines

- **Format**: PNG (recommended) or JPG
- **Naming**: `screenshot-N.png` where N is the number (1, 2, 3, etc.)
- **Size**: Ideally 1200-2000px wide for high-resolution displays
- **Content**: Clear, focused screenshots showing key features
- **WordPress.org Requirements**: 
  - Screenshots are displayed in order (1, 2, 3...)
  - First screenshot is the most important (featured)
  - Captions are defined in readme.txt

## Adding New Screenshots

1. Save screenshot as `screenshot-N.png` in this directory
2. Update `README.md` Screenshots section
3. Update `readme.txt` Screenshots section with caption
4. Ensure image is high quality and clearly shows the feature
